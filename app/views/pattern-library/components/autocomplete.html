Autocomplete

{% include "includes/experimental.html" %}

Type-ahead can be used to assist users in selecting from a known set of options. It can help constrain choices, where using a free-text field would produce to much variability in the user input. It can be used as an alternative to select lists in some situations.

Before using it

Consider whether there are other ways of collecting the information. If 90% of users are from the UK, you may decide to ask them if they are from the UK, and, only if they aren't, do you ask them where they are from.

When using it

Us it where the user is likely to know the answer ahead of time ir their country of birth.

Constrained and unconstrained input

Where the user "must" select from a constrained list, they should be clear that they need to make a selection.

Where the user doesn't "have" to select from a constrained list, type-ahead can help standardise inputs, whilst giving users the freedom to enter other choices. Eg familial relation - the majority of people are 'mother / father / brother / sister', but users can choose to enter less common types.
